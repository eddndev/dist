Achronyme
Scientific Operations Calculator

A functional programming language built in Rust for mathematical computing, digital signal processing, linear algebra, and scientific analysis.

Get Started GitHub
example.soc 
// Define a recursive factorial function
let factorial = rec n =>
  if(n <= 1, 1, n * factorial(n - 1))

// Vector operations with HOFs
let v = [1, 2, 3, 4, 5]
let squared = map(x => x^2, v)
let total = reduce((a, b) => a + b, 0, squared)

// Complex number support
let z = 3 + 4i
let magnitude = abs(z)  // 5.0

// Pattern matching
let describe = n => match n {
  0 => "zero",
  1 => "one",
  _ => "many"
}

print(factorial(5))  // 120
print(total)          // 55

Why Achronyme?

Built for scientific computing with modern language features and a focus on mathematical operations.

Functional First

Immutable by default with first-class functions, closures, and powerful pattern matching.

Scientific Computing

Native support for complex numbers, tensors, linear algebra, statistics, and DSP.

Built in Rust

High performance and memory safety with a modern, expressive syntax.
